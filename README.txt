Steps to set-up and run the virtual classroom application on locla host:

1. Create a project folder and inside it unzip the client.zip and server.zip as client and server folders respectively.

2. In client folder download the dependencies by the following command.
	cd client
	npm i

3. In server folder download the dependencies by the following command.
	cd server
	npm i

4. The above steps should create nodemodules folders each inside client and server folders.

5. Run the application in development mode using the following command
	cd server
	npm run dev

The application would be running on localhost with server active on port 8080
